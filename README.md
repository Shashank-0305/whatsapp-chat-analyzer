# whatsapp-chat-analyzer

WhatsApp Chat Analyzer lets you upload any exported WhatsApp 
conversation (.txt file) and instantly see:

- Who messages the most
- When the chat is most active
- Which words and emojis are used the most
- How activity trends over time

Works for both **individual** and **group** chats.
Supports **English** and **Hinglish** (Hindi-English mixed) chats.
